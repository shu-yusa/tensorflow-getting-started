# TensorFlow tutorial
Learn TensorFlow by doing tutorial.

## Requirements
* `python3`
* `virtualenv`

## Setup
Create a virtualenv environement by
```bash
virtualenv env
source env/bin/activate
```

Install `TensorFlow`.
```bash
pip install tensorflow
```

## Run scripts
```bash
python <script-name>
```
